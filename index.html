Clancy's Multiverse Simulator

Exoplanets in the style of The Midnight Gospel (Watch from 0:41 in this video). Data Source: Kepler Objects of Interest table (cumulative), NASA Exoplanet Archive.

Continents are generated from randomly placed vertices. The number of vertices are the full earth days between planetary transits (orbital period).

Depth of atmosphere is a function of the temperature of the planet.

Red X is a FALSE POSITIVE; disproved as an exoplanet.

New planets generated on load. Kepler name if available.
